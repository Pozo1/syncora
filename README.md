# syncora
Syncora: Conexão Musical via Grafos  MVP que conecta usuários pelo que ouvem no Spotify em tempo real. Usa Interseção de Vizinhança e Jaccard para validar afinidade musical. Projeto para Teoria dos Grafos e Mobile (5º Semestre - CC).  Tech: Flutter, API Spotify e Firebase. Foco: Grafos dinâmicos e similaridade.
